Corona 18 (rev.2.2) software for "monitoring" and "measuring"
----------------------------------------------------------------------------------------------------
Installation:
- Check files list 
	data.cab
	InstMsi.exe
	InstMsiW.exe
	setup.exe
	setup.ini
	install.msi
	readme.txt
- Run setup.exe file and instal program
- Open properties window on Start/Programs/Corona18 rev.2 /Monitoring
  and change "Run in" property  to "c:\Program Files\Corona18 rev.2\" 
  or another if you changed Installation directory from default
- Open properties window on Start/Programs/Corona18 rev.2 /Measuring
  and change "Run in" property  to "c:\Program Files\Corona18 rev.2\" 
  or another if you changed Installation directory from default
- Run Start/Programs/Corona18 rev.2 /Monitoring or Measuring
------------------------------------------------------------------------------------------------------
User guide:
- Plug in the data cable
- Run program 
- Set right "Port name" (right bottom) 
- Create new data file - "New" button (right side)
- Click "Get new" button (right side) and wait some minutes
	look at "LED" in status string (bottom center)
		"LED" is green - Ok
		"LED" is red - check data cable or change "Port name"
- "Get new" button begin receiving new data process. 
- After exit program - data file name saved in file "c:\Program Files\Corona18 rev.2\settings.ini"
- In next program start data file will be loaded automatically
- You may change program settings to automatic new data receiving
	Set right interval between requests in minutes "dt" (right top)
	Click "LED" near "Auto" (right midle)
	When progres bar under "Auto" will be all blue begin receiving new data
- This settings also are saved in  "c:\Program Files\Corona18 rev.2\settings.ini"
