Corona 18+ software for "monitoring" only
----------------------------------------------------------------------------------------------------
Installation:
- Check files list 
	data.cab
	InstMsi.exe
	InstMsiW.exe
	setup.exe
	setup.ini
	install.msi
	readme.txt
- Run setup.exe file and instal program
- Open properties window on Start/Programs/Corona18+/Corona18+
  and change "Run in" to "C:\Program Files\Corona18+" 
- Run Start/Programs/Corona18+/Corona18+
------------------------------------------------------------------------------------------------------
User guide:
- Run program. Create new monitoring data file - "New" button (right side)
- Set right "Port name" (right bottom)
- Click "Get new" button (right side) and wait some times
	loock at "LED" in status string (bottom center)
		"LED" is green - Ok
		"LED" is red - check data cable or change "Port name"
- "Get new" button begin receiving new data process. 
- After exit program - data file name saved in file "C:\monset.ini"
- In next program start data file will be loaded automatically
- You may change program settings to automatic new data receiving
	Set right interval betvin reqwests in minutes "dt" (right top)
	Click "LED" near "Auto" (right midle)
	When progres bar under "Auto" will be all blue begin receiving new data
- This settings also are saved in  "C:\monset.ini"
